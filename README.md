# laptop-install

1. Install [Homebrew](https://brew.sh):

  ```
  /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  ```


2. Clone [laptop-install](https://github.com/dunnkers/laptop-install):

  ```
  mkdir -p ~/git
  git clone https://github.com/dunnkers/laptop-install.git ~/git/dunnkers/laptop-install
  ```

3. Run laptop install script:

  ```
  cd ~/git/dunnkers/laptop-install
  sh install.sh
  ```
